# Bayard Vacations AI Chatbot
This project is an AI-powered chatbot built using Python and Flask. 
It helps customers with travel queries such as destinations, packages, pricing, and booking.
## How to Run
1. Install dependencies:
   pip install -r requirements.txt
   
2. Start the Flask app:
   python app.py
   
3. Open in browser:
   http://127.0.0.1:5000
